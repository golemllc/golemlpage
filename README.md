# Golum AI Landing Page

Futuristic / quantum-era landing page for **Golum AI** — Tech Solutions & Blockchain.

## Run locally

1. Install Node.js (v18+)
2. Install deps:
   ```bash
   npm install
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```
4. Open: http://localhost:3000

## Deploy to Vercel
- Push this repo to GitHub
- Go to vercel.com → New Project → Import → Deploy
